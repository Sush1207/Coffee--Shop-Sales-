# Coffee--Shop-Sales-
Data analysis and visualization project for "Coffee Shop Sales" using Excel. Features data cleaning/transformation via Power Query and interactive dashboard creation using Pivot Tables.
**Data Cleaning & Transformation**: Utilized Power Query (Get & Transform Data) for efficient data ingestion, cleaning, transformation, and the creation of insightful new columns (e.g., peak hours, sales segments).
**Data Modeling**: Established relationships and structured the data model within Excel.
**Visualization Dashboard**: Developed a dynamic and interactive visualization dashboard using Pivot Tables and Pivot Charts to highlight key sales trends, product performance, and hourly demand.
